# 📚 タスク管理アプリ完全習得のための30日間ハンズオン教材

## 🎯 この教材について

本教材は、**プログラミング完全未経験者**が、現代のWebアプリケーション開発を体系的に学び、実際に動作する「タスク管理アプリ」を**ゼロから作り上げる**ための30日間の実践的ハンズオン教材です。

### 教材の特徴

- ✅ **完全自己完結型**: すべてのコードを教材内に掲載。リポジトリを見に行く必要はありません
- ✅ **徹底的な例え話**: 専門用語は必ず日常の例えで解説
- ✅ **細かいステップ分割**: 初心者が迷わないよう、1ステップ1アクション
- ✅ **3回推敲済み**: すべての記事が厳格な品質管理プロセスを経て作成

### 技術スタック

この教材で学ぶ技術:

- **フロントエンド**: React 18, Next.js 15 (App Router), TypeScript 5.6
- **UI フレームワーク**: Material-UI 6.4
- **バックエンド**: tRPC 11.6 (型安全なAPI通信)
- **データベース**: PostgreSQL + Prisma 6.16 (ORM)
- **認証**: NextAuth.js 4.24
- **テスト**: Vitest 3.0, Playwright
- **開発ツール**: Docker, Task (タスクランナー)

### 完成するアプリの機能

30日後、あなたは以下の機能を持つアプリを作成できるようになります:

1. **ユーザー認証機能**
   - 新規登録・ログイン
   - パスワードのハッシュ化
   - セッション管理

2. **プロジェクト管理機能**
   - プロジェクトの作成・編集・削除
   - プロジェクトメンバー管理
   - プロジェクトの色分け・アーカイブ

3. **タスク管理機能**
   - タスクのCRUD操作
   - ステータス管理（TODO、進行中、レビュー中、完了など）
   - 優先度設定（低、中、高、緊急）
   - タイマー機能（作業時間計測）
   - タスクの担当者設定

4. **コメント機能**
   - タスクへのコメント投稿
   - コメントの編集・削除
   - ユーザー権限に基づく操作制限

5. **検索・フィルタリング機能**
   - 全文検索
   - ステータス・優先度でのフィルタリング
   - 担当者でのフィルタリング

6. **レポート機能**
   - ダッシュボード（統計情報表示）
   - 週次レポート
   - チャート表示

7. **ユーザー管理機能**
   - ユーザー一覧・詳細表示
   - プロフィール編集
   - パスワード変更
   - 管理者権限

## 🎓 学習の進め方

### 推奨学習スケジュール

- **1日あたりの学習時間**: 2〜3時間
- **実践時間**: 各Dayの実装に1.5〜2時間
- **復習時間**: 30分〜1時間

### 学習のコツ

1. **コードは必ず自分で打つ**: コピー&ペーストだけでは身につきません
2. **わからない部分は飛ばさない**: 疑問点は必ずその日のうちに解決
3. **エラーを恐れない**: エラーは成長のチャンス
4. **毎日少しずつ**: 一気にやるより、毎日継続が大切
5. **完璧を求めない**: 60%の理解で次へ進み、後で戻って理解を深める

### サポート体制

- 各Dayの最後に「理解度チェック」で復習
- よくあるエラーと解決方法を記載
- 質問は Issues に投稿可能（準備中）
