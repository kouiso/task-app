# Curriculum Style & Voice (教材文体スペック)

## 基調 — ですます体 + 軽い語りかけ

地の文（コードブロック・引用以外）はすべて **ですます体** で統一する。

| 許可される語尾 | 禁止される語尾 |
|---|---|
| です / ます / ました / でした | だ / だぞ / だろ / しろ |
| しましょう / してみましょう | やで / やな / やねん / ちゃう |
| ですね / ますね | ワイ / からや / ええわけやない |
| ポイントです / 確認してみましょう | (命令形 alone: 「書け」「使え」「確認しろ」) |

**温度感の目安**: 読者（学習者）に話しかける親しみは持つが、タメ口・関西弁にはしない。
「〜してみましょう」「ここがポイントです」「確認してみましょう」程度の語りかけが適切。

---

## 禁止カテゴリ 1 — 関西弁・タメ口

### NG → OK 対比例（実際の教材から）

| ❌ NG (実ファイル例) | ✅ OK |
|---|---|
| `初期セットアップは完了やで。` | `初期セットアップが完了しました。` |
| `サーバー側がまだないからや。` | `サーバー側がまだないためです。` |
| `最初の一枚目は、ただ映えればええわけやない。` | `最初の一枚目は、見栄えだけが目的ではありません。` |
| `今日はここを超えるで。` | `今日はここを乗り越えましょう。` |
| `ワイが書いたコードを見てみよか。` | `実際のコードを見てみましょう。` |

**検出パターン（check_tone.py が捕捉）**:
- 文末の `やで`、`やな`、`からや`、`ちゃう`、`やねん`、`ええわけやない`
- 一人称 `ワイ`

## 禁止カテゴリ 2 — AI典型構文

banned-phrases.md の教材向けサブセット。地の文に出現したら即修正。

| ❌ NG | ✅ OK |
|---|---|
| `することができます` | `できます` |
| `これにより` | `だから` / `なので` / `よって` |
| `〜と言えるでしょう` | `〜です` / `〜になります` |
| `〜ではないでしょうか` | `〜ですよね` / `〜だと思います` |
| `いかがでしたでしょうか` | (削除する) |
| `ここでは〜について解説します` | (削除して本文から始める) |
| `さまざまな場面で` | (削除するか具体例を書く) |
| `言うまでもありません` | (削除する) |
| `あらゆる観点から` | (削除するか具体項目を列挙) |

### NG → OK 対比例

| ❌ NG | ✅ OK |
|---|---|
| `tRPC を使うことができます。` | `tRPC が使えます。` |
| `これにより型安全が実現します。` | `なので型安全が実現します。` |
| `活用すると言えるでしょう。` | `活用できます。` |

---

## 禁止カテゴリ 3 — 英語直訳調

機械翻訳由来の定型表現。教材の日本語として不自然なため禁止。

| ❌ NG | ✅ OK |
|---|---|
| `〜することは可能です` | `〜できます` |
| `〜について言及` | `〜について` / `〜を紹介` |
| `〜を実施します` | `〜します` / `〜を行います` |
| `〜に関しても同様です` | `〜も同じです` |

### NG → OK 対比例

| ❌ NG | ✅ OK |
|---|---|
| `認証を実施することは可能です。` | `認証できます。` |
| `セッションについて言及しました。` | `セッションを説明しました。` |

## 適用スコープと確認方法

このスペックは `material/**` 配下のすべての `.md` ファイルに適用される。

教材を書いたあと・修正したあとに必ず実行：

```bash
python3 scripts/curriculum-qa/check_tone.py material/30days-curriculum/<day_file>.md
```

`check_quality.sh` の Step 7 でも自動実行される。

## セルフチェック（書く前の3問）

1. この語尾は「ですます」か？ — タメ口・関西弁になっていないか
2. `することができます` / `これにより` / `〜と言えるでしょう` を使っていないか
3. 「〜することは可能です」等の英語直訳調になっていないか

**3問すべて OK なら、そのまま書いてよい。**
