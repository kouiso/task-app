## GitHub Copilot Agent Skills

このプロジェクトは、GitHub Copilot Agent Skillsに対応しています。

Copilotを使用する際、プロジェクト固有のルールや定型作業を自動的に認識し、適切なサポートを提供します。

利用可能なスキル:
- **create-learning-materials**: 学習教材の作成
- **follow-coding-guidelines**: コーディング規約の遵守
- **fix-lint-errors**: リント・フォーマットエラーの修正
- **update-documentation**: ドキュメントの更新

詳細は [.github/skills/README.md](./.github/skills/README.md) を参照してください。
