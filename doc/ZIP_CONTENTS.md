## Required verification

```sh
bash scripts/build-zip.sh
unzip -l task-app-curriculum-v1.1.zip | grep 'task-app/prisma/migrations/.*/migration.sql'
rm -rf /tmp/task-app-curriculum-v1.1-verify
mkdir -p /tmp/task-app-curriculum-v1.1-verify
unzip -q task-app-curriculum-v1.1.zip -d /tmp/task-app-curriculum-v1.1-verify
cd /tmp/task-app-curriculum-v1.1-verify/task-app

npm run build
npm test
```

`npm test` requires the test PostgreSQL endpoint from `.env.example` or an equivalent reachable database.
